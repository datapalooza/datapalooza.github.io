Sharing data and notebooks can be very useful for team collaboration.

Exporting Data and Notebooks

  1. Go to Data Scientist Workbench and click the My Notebooks menu.
  2. Find the file or notebook you want to share, and click the twistie () to the left of it.

  3. Click the share button ().

  4. Click the Show Link link to get the link to the shared notebook.

  5. Send this link to the person you're sharing the file or notebook with.

Importing Data and Notebooks

  1. Copy the link of the file or notebook you want to import (from step 5 above).
  2. Go to Data Scientist Workbench and click the My Notebooks menu.

  3. Paste the link in the text field at the top right corner of the page.

  4. Hit Enter to import the file or notebook.
